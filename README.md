# Flask REST API

## 📌 Description
This project is a simple REST API built using Flask.
It performs CRUD operations on user data.

## 🚀 Features
- GET users
- POST user
- PUT update user
- DELETE user

## 🛠️ Tech Used
- Python
- Flask

## ▶️ How to Run
pip install flask
python app.py

## 📡 API Endpoints
GET /users
POST /users
PUT /users/<id>
DELETE /users/<id>

## 🧪 Testing
Tested using Postman
